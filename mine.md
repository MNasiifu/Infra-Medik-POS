To activate: In Supabase dashboard → Edge Functions → deploy efris-submit, then Settings → Edge Function Secrets → add EFRIS_API_URL, EFRIS_DEVICE_NO, EFRIS_DEVICE_SECRET, EFRIS_TIN once URA provides sandbox credentials.

Before production: Convert public/icons/icon.svg → icon-192.png and icon-512.png (use any SVG→PNG tool). The SVG already works as a fallback in modern browsers.
